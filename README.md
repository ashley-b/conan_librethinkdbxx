# Conan librethinkdbxx recipe

## Description

Simple Conan recipe for https://github.com/AtnNn/librethinkdbxx

This driver is compatible with RethinkDB 2.0. It is based on the official RethinkDB Python driver.

## Useage

Example conanfile.txt snippet for consumers

```
[requires]
librethinkdbxx/cci.20171109
```
